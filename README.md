# SwiftBus

SwiftBus is a React + Vite bus ticket booking interface.

## Features

- Route search with city autocomplete
- Date picker and seat availability
- Interactive seat selection by layout
- Passenger checkout with validation
- Ticket generation with QR code
- Booking history with cancel and print
- Persistent state via localStorage

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
